# Forever - Fashion E-Commerce Website

A modern and responsive fashion e-commerce website built with **React**, **Vite**, and **Tailwind CSS v4**. The application allows users to browse clothing collections for **Men**, **Women**, and **Kids**, search products, filter collections, and view detailed product information.

---

## Overview

Forever is a front-end e-commerce application designed with a clean and minimal user interface. It focuses on providing an intuitive shopping experience with responsive layouts, product filtering, and reusable React components.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/forever-ecommerce.git
```

---

### Navigate into the Project

```bash
cd forever-ecommerce
```

---

### Install Dependencies

```bash
npm install
```

---

### Start Development Server

```bash
npm run dev
```

---

### Build for Production

```bash
npm run build
```

---

### Preview Production Build

```bash
npm run preview
```
